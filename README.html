# Learn SQL This is a repository for anyone who wants to learn SQL.
It's an effort by Yash and Aniruddha to understand and learn "SQL".
You can use all the queries laid down in this tutorial in the SQL Tryit Editor by W3Schools.

This is the Data Base we're using from https://www.w3schools.com/
You can also practice on Oracle offical website dedicated wholly to SQL from here Live SQL
On some points of the tutorial Oracle or PL/SQL Clauses and Fucntions were used. So you can download Oracle Express Edition 11g which does not requir any connection with SQL Server Client and made for the use of developers and learners, from here or there is a online interpreter provided by TutorialsPoint with online Oracle XE 11.2.0-1.0 all built in, which include some basic tables in it which you can use to practice on, from here. The Tables that I found were there were DEPT, EMP, DUAL, SYSTEM_PRIVILEGE_MAP, TABLE_PRIVILEGE_MAP, STMT_AUDIT_OPTION_MAP, AUDIT_ACTIONS.
